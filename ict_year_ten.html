Qualities of Good Information

The qualities of good information are:
1. Accuracy - it must be free from errors.
2. Reliability - it should be from a reliable source.
3. Relevance - it must be suitable for the purpose it is required for.
4. Accessibility - it should be easy to obtain or access when needed.
5. Timely - it should be available at the right time.
6. Completeness - it should be comprehensive and contain all the necessary details.

Activities involved in processing data into Information

Data collection - gathering data from various sources.
Classification or sorting - identifying certain characteristics in items of data and categorizing or grouping them according to those characteristics. Coding
Summarizing.
Input - entering the data using input devices into the computer for processing.
Data processing, calculating or editing: The process of converting the data into information.
Validation.
Translating - changing the language of data into another.
